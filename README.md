# Dog Store

Monorepo with:
- `frontend/`: React + TypeScript (RTL by default)
- `backend/`: Express + TypeScript API

## Prerequisites
- Node.js (npm)

## Setup
From the repo root:
```bash
npm install
```

## Run (dev)
```bash
npm run dev
```

- Frontend: http://localhost:5174
- Backend API: http://localhost:5000

## API
- `GET /api/foods` (placeholder menu data)
